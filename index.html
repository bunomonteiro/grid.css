Grid.css

A simple, responsive, mobile first and and without dependencies 16-columns grid system. See a demo: Grid Demo

Don't worry about the size. Minified and gzipped is less than 3.5kb.

Version

1.0.0

Maintained by

Bruno Monteiro, also known as b'uno. (linkedin - hashcode - twitter - facebook)

Tech

Grid.css was created using a number of open source projects:

  • Node.js - evented I/O for the backend
  • Gulp - the streaming build system
  • Sass - syntactically awesome style sheets

And of course Grid.css itself is open source with a public repository.

How to use

Add the following meta tag to your HTML’s

section to keep viewport scale: 
<meta name="viewport" content="width=device-width,initial-scale=1">

Add the following stylesheet to your HTML’s

section: 
<link rel="stylesheet" href="YOUR_STYLE_PATH/grid.min.css">

or if you want to use the bundle that includes the reset.css by Eric A. Meyer (recommended)

<link rel="stylesheet" href="YOUR_STYLE_PATH/grid.bundle.min.css">

Then the markups:

<div class="container">
  <div class="row">
    <div class="col xs-16">
      ...
    </div>
  </div>
</div>

Development

Want to contribute? Great!

Grid.css uses Gulp + Browsersync for fast developing. Make a change in your file and instantanously see your updates!

Open your favorite Terminal and run these commands.

You need Gulp installed globally:

$ npm i -g gulp

then:

$ git clone https://github.com/bunomonteiro/grid.css.git grid
$ cd grid
$ gulp serve

Customize

16 columns is too much or too little for you? Don't worry, just change the variable at the beginning of src/styles/grid.scss file:

$total-cols: 16 //default

Do you want to increase or decrease the margins?

$col-margin: 2% //default

After that:

$ gulp clean
$ gulp
